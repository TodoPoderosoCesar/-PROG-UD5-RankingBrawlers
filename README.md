# Brawler Ranking

Está app dispone de un sistema muy sencillo para organizar en base el rango en estrellas (de 1 a 5) el desempeño que tiene los Brawlers.
Por otro lado, según vayan saliendo nuevos personajes se dispone de la posibilidad de añadirlos, a su vez tambien de eliminarlos e incluso de editarlos.
